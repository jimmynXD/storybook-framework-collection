# @local/prettier-config

Add prettier code formatting rules to the root `package.json.

## Installation

Add the following to the root `package.json` file:

```json
{
  "devDependencies": {
    "@local/prettier-config": "workspace:*",
    "prettier": "^2.8.8"
  },
  "prettier": "@local/prettier-config"
}
```
